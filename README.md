# Iron-Carbon Phase Transformation Simulator

This repository contains a Python-based phase transformation simulator developed for studying microstructural evolution in the Iron-Carbon (Fe-C) phase diagram.

The simulator allows users to provide simple inputs such as:

* Temperature profile
* Carbon percentage

Based on these inputs, the program generates a qualitative simulation of phase transformations and microstructural evolution during thermal processing.

The project is designed primarily for educational and student learning purposes to help visualize and understand phase transformation behavior in steels.

## Project Objective

The goal of this project is to provide an interactive and simplified computational tool for understanding:

* Iron-Carbon phase transformations
* Microstructural evolution

The simulator is not intended for industrial-grade material prediction or accurate metallurgical design. Instead, it aims to mimic realistic transformation behavior to support conceptual understanding and academic learning.

---

## Features

* User-defined temperature profiles
* Adjustable carbon composition
* Visualization of phase transformation behavior
* Microstructural evolution simulation
* Python-based implementation

---

## Methodology

The simulator uses transformation curve equations derived through Least Squares Curve Fitting

Empirical transformation relationships for the Iron-Carbon phase diagram were mathematically approximated to reproduce realistic transformation trends.
